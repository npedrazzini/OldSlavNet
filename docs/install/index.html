Install OldSlavNet

Get OldSlavNet

Clone OldSlavNet’s GitHub repository and download the relevant binaries.

1. Clone OldSlavNetâ€™s repository

Run in your terminal:

git clone https://github.com/npedrazzini/OldSlavNet

Or only download the relevant python files from the repository:

  • parser.py
  • oldslavdep.py
  • learner.py
  • decoder.py
  • mnnl.py
  • utils.py

2. Download the relevant LFS binaries:

Download the model and model parameter files from here and add them to your working directory:

  • model
  • model.params

3. Install required dependencies:

OldSlavNet runs on Python 3.4+ or 2.7 and requires:

  • pip>=19.0.3
  • cython>=0.29.21
  • numpy>=1.19.1
  • dynet==2.0

You can run:

pip install -r requirements.txt

If the above does not work, install each of the above individually.

  • Upgrade pip: pip install --upgrade pip
  • Install cython and numpy: pip install cython numpy
  • Install dynet: here are detailed instructions on how to install it.
