Pre-process your data

Convert your text file to CoNLL-U

Assuming you have an already tokenized and sentencized (i.e. once sentence per line) text file (corpus.txt), you can now run the convert.py script (which you can find here) to get the 10-column CoNLL-U file required by OldSlavNet as input:

python convert.py < corpus.txt > corpus.conllu
