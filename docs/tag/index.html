Annotate your texts

From the ROOT directory run:

./tag.sh

You will be prompted to enter:

  1. The name of the model you want to use to tag your texts (press Enter for ‘OldSlavNet’)
  2. The name of the text you want to annotate (press Enter for ‘everything inside the folder ./test_data/tobeannotated/').

Your tagged texts will be saved under ./test_data/annotated/ with the same name as the original text:
