Run OldSlavNet

You can now add syntactic annotation using OldSlavNet’s pretrained model by running the following:

python parser.py \
--predict \
--model /path/to/model \  
--params /path/to/model.params \ 
--test /path/to/input/file/corpus.conllu \
--outdir /path/to/output/directory \
--output output-name.conllu

Where:

  • --model: Path to model file (model). \
  • --params: Path to model hyper-parameters file (model.params).\
  • --test: Path to CoNLL-U input file.\
  • --outdir: Path to directory where your output file will be saved.\
  • --output: Choose a name for your output file.
